# Card Shuffle Suite
A collection of card shuffling techniques in a variety of languages, frameworks, and tests

## Currently Implemented
* Javascript
  * Random Sort
  * Riffle
  * Cut
  * Fisher-Yates (aka Knuth) Shuffle
* Demo Runner (for JavaScript code)
